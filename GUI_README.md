# LeRobot SO100 GUI 工具使用说明

这是一个基于 PySide6 的图形化界面工具，用于简化 `RUN_GUIDE_CN.md` 中各种命令的执行。

## 安装依赖

首先需要安装 PySide6：

```bash
pip install PySide6
```

## 运行 GUI

在项目根目录下运行：

```bash
python lerobot_gui.py
```

## 功能模块

GUI 包含以下 6 个标签页，对应 `RUN_GUIDE_CN.md` 中的各个步骤：

### 1. 环境准备
- 克隆仓库
- 创建 Conda 虚拟环境
- 安装依赖包

### 2. 硬件连接与校准
- 查找串口
- 设置电机 ID（Robot 和 Teleop）
- 校准机器人
- 查看摄像头

### 3. 遥操作
- 配置机器人、Teleop 和摄像头参数
- 开始遥操作测试

### 4. 数据录制
- 配置机器人、Teleop、摄像头和数据集参数
- 开始数据录制

### 5. 训练
- 配置数据集、策略和输出参数
- 开始训练或继续训练

### 6. 评估与回放
- 策略评估：使用训练好的策略进行评估
- 数据回放：回放已录制的数据

## 使用说明

1. **命令输出**：每个标签页底部都有一个输出区域，显示命令执行的结果
2. **参数配置**：在运行命令前，请确保填写所有必要的参数
3. **后台执行**：命令在后台线程中执行，不会阻塞 GUI
4. **清空输出**：可以随时点击"清空输出"按钮清除输出内容

## 注意事项

- 某些命令（如创建 Conda 环境）可能需要手动激活环境后再执行后续步骤
- Windows 系统下，串口路径格式可能不同（如 `COM3` 而不是 `/dev/ttyACM0`）
- 确保已正确安装所有依赖，特别是硬件相关的依赖（如 `feetech`）
- 训练和评估命令需要 GPU 支持，请确保 CUDA 环境配置正确

## 故障排查

如果遇到问题：
1. 检查命令输出区域中的错误信息
2. 确认所有参数都已正确填写
3. 确保 Python 环境和依赖都已正确安装
4. 对于硬件相关命令，确保设备已正确连接
